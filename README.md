# Comandos Para rodar a aplicação Localmente

### Clonar repositorio
```bash
git clone https://github.com/GMatheusst/School_App
```

### Localizar repositorio
```bash
cd School_App
```

### Baixar Dependencias do laravel
```bash
composer i
```

### Gerar as tabelas
```bash
php artisan migrate
```

### Gerar as chaves

```
php artisan key:generate
```

### Ligar as filas

```bash
php artisan queue:work

```

### Ligar o servidor
```
php artisan serve
```
